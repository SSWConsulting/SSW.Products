  ## YakShaver Now Supports .mov Files


  YakShaver now works with videos recorded using the MacOS Screenshot App,
  thanks to new .mov file extension support.


  ***


  ## Supported Recording Tools


  * Screenshot App (MacOS default)

  * SnagIt

  * Other tools that produce .mov or other supported formats (see our [setup
  guides](/docs))
---

With the addition of .mov file extension support, YakShaver now supports the default MacOS recording tool, Screenshot App!

Known tools that are currently supported:

* [Screenshot App](https://yakshaver.ai/docs/recording-work-item-screenshot-app)
* [SnagIt](https://yakshaver.ai/docs/setting-up-other-recording-tools)
* Other recording software that creates .mov or other video extension files (see the FAQ)
