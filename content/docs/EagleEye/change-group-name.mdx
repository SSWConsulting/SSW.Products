After you've deployed Eagle Eye, sometimes you want to change which email group you want to scan. Link to the docs that talks about setting it up initially, about how to select your group name and whatnot. This guide is supposed to help someone to navigate the Azure portal, find the correct space, reset their environment variable to something different, find their list of groups so they know what value to put in, and then restart the API and test that it's working.
