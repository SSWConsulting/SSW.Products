## Step 1: Create a New Tag

1. Go to [Portal | Tags](https://portal.ssweagleeye.com/tags) page.
2. Click on the **"New Tag"** button.
3. Fill out the required details in the form.
4. Click **"Create"** to save the tag.

> **Tip for AI Tags**: Ensure you test the prompt and response to verify the tag is working as expected.

## Step 2: Sync Data

### Scheduled Scan

A daily scheduled scan automatically processes the previous day's emails. The Power BI report refreshes daily to reflect the updated data.

### Manual Scan

If you want to see the tag in action immediately, you can create a manual job to scan emails.

1. Go to [Portal | Jobs | Manual](https://portal.ssweagleeye.com/jobs/manual) page.
2. Click the **"New Job"** button.
3. Select the desired **date range** and **tags** to scan.
4. Click **"Create"** to start the job.
5. Refresh the Power BI report to see the updated data.
