This guide walks you through connecting your TimePro tenant to Microsoft Dynamics 365. Once configured, TimePro can pull clients, projects, and users from Dynamics 365, and display your Dynamics 365 appointments in the weekly timesheet view.

## Open Admin and Go to Dynamics 365

From your TimePro tenant - e.g. \`northwind.sswtimepro.com\`:

1. click Admin in the menu
2. click Dynamics 365

Figure: Where to find Dynamics 365 integration

## Setup your Dynamics 365 Configuration

If Dynamics 365 is not yet configured, you will need to add a connection. If it is already configured, click Edit to update the settings:

1. Dynamics 365 Connection Type - choose between Active Directory (OnPrem) for on-premises Dynamics 365 or Client Secret (Cloud) for Dynamics 365/Dataverse in the cloud
2. Dynamics 365 Url - Enter your Dynamics 365 URL - e.g. \`northwind.crm.dynamics.com\`
