## üîç How It Works

TimePro is a **web app** - no desktop install. Once you [create a tenant](https://sswtimepro.com/docs/create-tenant) you're ready to go. Below is a list of the main services that you can manage in TimePro:

| **Service**      | **Details**                                                                                                                                                                        |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Timesheets**   | Log time by project, task, and client. Optional automation from Azure DevOps (and soon GitHub) so developers spend less time filling forms                                         |
| **Invoicing**    | Turn approved time and expenses into client invoices. Support for prepaid/discount strategies and credit notes                                                                     |
| **Integrations** | Sync with Xero for invoicing and receipts, Microsoft Dynamics 365 (CRM) for projects and clients, Azure DevOps for work items and timesheet hints, and Ingram Micro where relevant |
| **Reporting**    | Built-in reports for timesheets, invoices, tasks, and expenses, plus Power BI for ad hoc analysis                                                                                  |
| **Leave**        | EasyLeave module for leave requests, approvals, calendar sync (Outlook/CRM), and automatic leave timesheet entries                                                                 |

## üöÄ Key Features

* **Quick timesheet entry** - Optimized for fast, daily time entry so developers and consultants can stay accurate without the overhead.
* **Azure DevOps integration** - Pull in work items, commits, and check-ins to pre-fill or suggest timesheet entries.
* **Xero sync** - Push invoices and receipts to Xero for accounting and compliance.
* **CRM integration** - Keep clients and projects in sync with Microsoft Dynamics 365 for invoicing and project tracking.
* **Prepaid and credit handling** - Support for prepaid packages, discounts, and credit notes so invoicing matches how you sell.
* **Multi-tenant** - One platform, many organizations; each tenant gets isolated data and configuration.
* **EasyLeave** - Request leave, get approvals, and have leave time and calendar events updated automatically.
* **Reporting and Power BI** - Standard reports plus the flexibility to build your own views and dashboards.
