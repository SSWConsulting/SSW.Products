# Install YakShaver Browser Extension

1. Visit the [YakShaver landing page](https://yakshaver.ai)
2. Click on "Try it now" button
3. Download and add the extension to your Chromium browser (Chrome / Edge)
4. Allow any required permissions

![](/YakShaver/Docs/YakShaver-Try-it-now.png)\
**Figure: YakShaver extension download page with "Try it now" button**

### Using the Browser Extension

ðŸ’¡ Important: Navigate to the webpage you want to record before starting. Browser system pages (like chrome://extensions) don't support screen recording.

1. Click on the YakShaver icon in your browser toolbar
2. Click "Start Recording" to begin capturing your screen
3. Select which screen or application to share when prompted

![](/YakShaver/Docs/YakShaver-Start-Recording.png)\
**Figure: YakShaver recording interface with "Start Recording" button**

### Recording Process

1. Perform the task you want to document
2. Click "Stop Recording" when finished
3. YakShaver will automatically process your recording and create a ticket
4. You can click on open ticket to be redirected to your newly created ticket

### Troubleshooting

**Can't see the record button?** -\
Try refreshing the page or reinstalling the extension

**Want to cancel a task during recording?** -\
Simply click on the cancel button
