Learn how to configure custom system prompts in YakShaver to tailor AI-generated PBIs to your team's requirements.

## Configure System Custom Prompt in YakShaver Portal

YakShaver supports custom system prompts for PBI generation, allowing you to customize AI-generated content to match your team's specific requirements.

Follow these steps to configure custom system prompts:

### Prerequisites

You need admin permission for your tenant. Only admin accounts can edit custom system prompts.

### Accessing the Portal

Access the YakShaver portal using one of these methods:

1. Clicking the "Portal" button in the top-right corner of the [yakshaver.ai](https://yakshaver.ai) home page
2. Or directly visiting [https://portal.yakshaver.ai/](https://portal.yakshaver.ai/)

![Open Portal](/YakShaver/Docs/open-portal.png)
*Figure: Access the YakShaver portal from the home page*

### Configuration Levels

YakShaver supports two configuration levels for custom prompts: **tenant level** and **project level**.

#### Project Level Configuration

Users can also configure specific project level prompts to override the tenant settings. Admin accounts can click on an individual project to enter the project details, then modify the prompt in the "Custom System Prompt" field.

![Edit Project Prompt](/YakShaver/Docs/edit-project-prompt.png)
*Figure: Edit project level prompt configuration*

Regular users can view each project's prompt in the project list page.

![View Project Prompt](/YakShaver/Docs/view-project-prompt.png)
*Figure: View project level prompts in the project list*
