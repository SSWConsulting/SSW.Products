要开始通过 YakShaver 在您的 Azure DevOps 仓库中创建 shaves，请按照以下步骤操作。

> 注意：您必须同时拥有组织租户和 Azure DevOps 账户的管理员权限。

### 逐步说明

1. **将 Azure DevOps 连接到 YakShaver**\
   访问 [YakShaver 集成设置](https://portal.yakshaver.ai/settings/integrations) 并使用管理员账户授权。
2. 转到您想添加 YakShaver 的 **Azure DevOps 组织** 设置\
   您可以在此查看您所属的组织：[https://aex.dev.azure.com/me](https://aex.dev.azure.com/me) 地址应类似于 [https://dev.azure.com/\[yourOrg\]/\_settings](https://dev.azure.com/NorthwindTestOrg/_settings)
3. 转到 **常规 | 用户 | 添加用户**，并添加以下内容：\
   **用户/服务主体**：YakShaver.ai (ID: `be9ecbe4-c377-43b8-a560-ab137326ebf2`)\
   **访问级别**：基础\
   **添加到项目**：包含 YakShaver 应访问以创建工作项的所有项目\
   **Azure DevOps 组**：项目贡献者
   ![](/YakShaver/Docs/add-devops-projects-add-yakshaver.png)
4. **对每个组织重复操作**\
   如果您管理多个 Azure DevOps 组织，请对每个组织重复步骤 2-3。
5. **在 YakShaver 中同步项目**\
   访问 YakShaver 的 [项目设置页面](https://portal.yakshaver.ai/settings/projects) 并同步您的 Azure DevOps 项目。您的更改现在应反映在门户中。
   ![](/YakShaver/Docs/add-devops-projects-sync-projects.png)

一切就绪——YakShaver 现已连接到您的 Azure DevOps 项目。
