如果您无法访问 Chrome 网上应用店（例如，您在中国），可以使用本指南手动安装扩展程序。

注意：此方法仅用于测试目的，需要启用开发者模式。为了获得最佳体验和自动更新，请从[Chrome 网上应用店](https://chromewebstore.google.com/detail/yakshaver/glcoihkbmomiblcbpnlkafcipcfnikjc)安装。

## 安装步骤

### 第一步：下载 CRX 文件

1. [下载 YakShaver 扩展 CRX 文件](https://yakshaver.ai/TryItNow/yakshaver-extension.crx)
2. 文件将保存到您的下载文件夹
3. Chrome 会尝试自动安装扩展并显示无法安装的通知，但别担心，我们将在接下来的步骤中手动安装

### 第二步：打开 Chrome 扩展管理

1. 打开 Chrome，在地址栏输入 `chrome://extensions`，将打开 Chrome 扩展页面
2. 确保侧边栏选择了“我的扩展程序”
3. 在右上角启用“开发者模式”

![](/YakShaver/Docs/extension-dev-mode-v2.png)

**图示：Chrome 扩展页面 | 启用“开发者模式”**

### 第三步：安装扩展

1. 现在，进入保存 YakShaver 扩展 CRX 文件的文件夹
2. 将该文件从那里拖放到 Chrome 扩展页面的任意位置
3. 最后，确认安装

![](/YakShaver/Docs/extension-drag-drop.png)

**图示：将扩展文件拖放到扩展页面**

安装成功后，您应该看到：

* YakShaver 列在 chrome://extensions/ 的扩展列表中
* 扩展状态显示为“已启用”

![](/YakShaver/Docs/extension-success-install.png)

**图示：扩展安装成功**
