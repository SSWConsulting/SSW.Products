## 1. YakShaver 支持所有语言吗？（不仅仅是英语）

是的，YakShaver 支持世界上大多数语言。[查看支持的 OpenAI 列表](https://github.com/openai/whisper?tab=readme-ov-file#available-models-and-languages)。

查看由录音创建的问题示例，分别为[中文](https://github.com/SSWConsulting/Northwind365/issues/162)和[法语](https://github.com/SSWConsulting/Northwind365/issues/168)。

***

## 2. 我需要公司内有认证人员使用 YakShaver 吗？

SSW 建议您的组织中有了解 YakShaver 的人员。他们可以作为您的第一联络点，以防需要更新配置或出现任何异常。SSW 提供免费认证课程，成功完成后您的组织将获得价值 25 美元的 50 个免费积分。[点击这里申请](https://yakshaver.ai/certifications)。

***

## 3. 什么时候*不*应该使用 YakShaver？

YakShaver 非常适合提供上下文并准确展示用户体验。在特定情况下，当您只需要进行小的**文本更改**（改写、措辞改进或语气调整）且明确知道哪个项目和人员负责执行时，最好手动创建问题或 PBI。

在这些情况下，请遵循 SSW 规则[“你是否要求将内容从 X 改为 Y 的小变更？”](https://www.ssw.com.au/rules/change-from-x-to-y)。这确保了清晰、意图明确，并促进与内容负责人更好的协作。

***

## 4. 我刚创建了一个工作项，这些按钮是做什么的？

其实很简单！

* **打开工作项：** 在您首选的浏览器中打开该问题
* **移动工作项：** 如果您在错误的项目中创建了工作项，可以将其移动到正确的项目
* **关闭工作项：** 如果您误创建了工作项，可以关闭该工作项

![](/YakShaver/Docs/Work-Item-Card.png)

## 5. 如果我在视频中没有提到项目名称，会发生什么？

如果 YakShaver 无法确定项目，或置信度较低，它会显示一个消歧义卡片，供您手动选择正确的项目。

![](/YakShaver/Docs/disambiguation-card.png)
**图示：消歧义卡片，供您选择项目**

***

## 6. 为什么我的工作项中有这么多人被抄送？

当创建工作项时，转录中提到的任何人都会自动加入对话。对于项目相关的工作项，主要联系人和所有项目成员也会被标记，以确保团队的可见性和一致性。

***

## 7. 为什么我的 GitHub 问题是由 YakShaver 创建，而不是我的账户？

要在您的 GitHub 账户下创建和编辑问题，您需要将 GitHub 账户连接到 YakShaver。这确保问题正确归属您，并赋予您完整的编辑权限。请参见[连接您的 GitHub 账户](/docs/connect-your-github-account)获取设置说明。

***

## 8. 如何彻底从 GitHub 问题中删除机密信息？

如果您不小心在通过 YakShaver 录制的 GitHub 问题中包含了敏感内容（例如密码、私人数据），**仅编辑问题是不够的**。GitHub 会保留完整的修订历史。请参见[如何彻底删除 GitHub 问题中的机密信息](https://www.ssw.com.au/rules/remove-confidential-information-from-github)。

***

## 9. 我的 Shave 被 YouTube 下架了，发生了什么？

您的视频可能因违反 YouTube 的[社区准则](https://support.google.com/youtube/answer/9288567?hl=en)而被移除。录制屏幕时，请注意不要显示机密或敏感信息，如个人信息、账户数据或私人通信。上传前审查录制内容并熟悉 YouTube 政策，有助于防止未来被移除。

## 10. [我的 Shaves](https://portal.yakshaver.ai/my-shaves) 页面中每个 shave 状态代表什么？

* 待处理：Shave 正在处理中

* 已完成：Shave 已成功处理并交付

* 错误 - Shave 处理时遇到错误，未能完成

* 已取消 - Shave 被用户手动取消

* 积分不足 - 由于积分不足，Shave 无法完成

* 回退 - 当用户初始意图未完成时，发送了回退邮件
