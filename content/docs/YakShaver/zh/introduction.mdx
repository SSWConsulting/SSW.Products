YakShaver 通过将您的屏幕录制转换为结构化文档，简化了工作项的创建。请选择您偏好的录制方式，并按照设置说明开始使用。为了获得最佳体验，我们建议从 Microsoft Teams 或浏览器扩展开始。

## Microsoft Teams 设置

Teams 集成提供无缝的协作功能，非常适合团队环境。

### 安装步骤

1. 在 Microsoft Teams 左侧边栏点击 **应用**
2. 在搜索框中搜索 “YakShaver”
3. 点击 **添加** 安装 YakShaver 机器人
4. 右键 YakShaver，选择“固定”以将其保留在侧边栏

![](/adding-bot.png)

**图示：将 YakShaver 机器人添加到 Teams**

### 初始配置

1. 进入 Teams 侧边栏的 YakShaver 机器人
2. 在聊天框输入 'on' 激活自动处理
3. 激活后您将看到确认卡片

![](/turn-on-card.png)

**图示：YakShaver 激活确认**

### 可用命令

* `on` - 设置 YakShaver 始终处理您的视频
* `off` - 设置 YakShaver 处理前询问
* `status` - 查看当前设置
* `leaderboard` - 查看排行榜
* `help` - 显示命令列表
* `hi` - 显示欢迎信息

📖 准备在 Teams 中录制？请参阅我们的 [Teams 录制指南](/docs/recording-work-item-teams)

## 浏览器扩展设置

浏览器扩展支持 Chrome 和 Edge 浏览器的无缝集成。

### 安装步骤

1. 访问 [YakShaver 主页](https://yakshaver.cn) 并点击“立即试用”
2. 将扩展添加到 Chrome/Edge
   ![](/YakShaver/Docs/Try-it-now.png)

**图示：安装并固定 YakShaver 浏览器扩展**

3\. 将其固定到浏览器工具栏，方便访问

4\. 按提示允许所需权限

💡 **重要提示：**

* 开始前请确保您处于可录制页面（浏览器系统页面如 chrome://extensions 不支持）
* 为获得最佳效果，请确保麦克风权限已启用
* 扩展在 Chrome 或 Edge 浏览器中表现最佳

📖 准备开始录制？查看我们的详细 [浏览器扩展录制指南](/docs/recording-work-item-browser-extension)

## 第三方工具设置

YakShaver 支持多种录制工具，适合偏好现有工作流程或需要特定功能的用户。

### 支持的工具

* **[Snagit](/docs/recording-work-item-snagit)** - 专业屏幕捕获工具（推荐 Windows 用户）
  * 提供高级编辑功能
  * 支持自定义快捷键
  * 适合高级用户
* **[Windows 截图工具](/docs/recording-work-item-snipping-tool)** - Windows 内置工具
  * 无需额外软件
  * 快速简便
  * Windows 键 + Shift + R 快捷键
* **[Mac 截图应用](/docs/recording-work-item-screenshot-app)** - MacOS 内置工具
  * 原生 MacOS 集成
  * CMD + Shift + 5 快捷键
  * 无需额外软件

### 通用设置流程

1. 在 OneDrive 中创建专用文件夹：
   * Snagit: `OneDrive/Recordings/Snagit/`
   * 截图工具: `OneDrive/Recordings/SnippingTool/`
   * 截图应用: `OneDrive/Recordings/ScreenshotApp/`
2. 配置工具将录制内容保存到相应文件夹
3. 确保麦克风设置正确
4. 测试录制以验证设置
