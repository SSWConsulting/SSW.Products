YakShaver 通过将你的屏幕录制内容转化为结构化文档，从而简化工作事项的创建流程。请选择你偏好的录制方式，并按照设置指南开始使用。为了获得最佳体验，我们建议从 Microsoft Teams 或浏览器扩展开始。

## Microsoft Teams 设置

Teams 集成提供无缝的协作功能，非常适合团队环境。

### 安装步骤

1. 在 Microsoft Teams 中，点击左侧边栏的 **应用**
2. 在搜索框中搜索 "YakShaver"
3. 点击 **添加** 来安装 YakShaver 机器人
4. 右键点击 YakShaver 并选择“固定”将其保留在侧边栏

![](/adding-bot.png)

**图示：将 YakShaver 机器人添加到 Teams**

### 初始配置

1. 在 Teams 的侧边栏中打开 YakShaver 机器人
2. 在聊天框中输入 `on` 激活自动处理功能
3. 当 YakShaver 启动时，你会看到一张确认卡片

![](/turn-on-card.png)

**图示：YakShaver 激活确认**

### 可用指令

* `on` - 让 YakShaver 始终处理你的视频
* `off` - 每次处理前询问是否确认
* `status` - 查看当前设置状态
* `leaderboard` - 查看排行榜
* `help` - 显示指令列表
* `hi` - 显示欢迎信息

📖 准备好在 Teams 中开始录制了吗？查看我们的 [Teams 录制指南](/docs/recording-work-item-teams)

## 浏览器扩展设置

浏览器扩展可无缝集成至 Chrome 和 Edge 浏览器。

### 安装步骤

1. 访问 [YakShaver 官网](https://yakshaver.ai) 并点击“立即试用”
2. 将扩展程序添加到 Chrome 或 Edge
   ![](/YakShaver/Docs/Try-it-now.png)

**图示：安装并固定 YakShaver 浏览器扩展**

3. 将其固定到浏览器工具栏，便于访问

4. 当出现权限提示时，请允许相关权限

💡 **重要提示：**

* 请确保你当前页面支持录制（如 chrome://extensions 这类系统页面不支持）
* 为获得最佳效果，请启用麦克风权限
* 此扩展在 Chrome 或 Edge 浏览器上体验最佳

📖 想开始录制？查阅我们的 [浏览器扩展录制指南](/docs/recording-work-item-browser-extension)

## 第三方工具设置

YakShaver 支持多种录屏工具，适合偏好自定义工作流或有特定需求的用户。

### 支持的工具

* **[Snagit](/docs/recording-work-item-snagit)** - 专业的屏幕捕捉工具（推荐 Windows 用户）
  * 提供高级编辑功能
  * 支持自定义快捷键
  * 非常适合高级用户
* **[Windows 截图工具](/docs/recording-work-item-snipping-tool)** - Windows 内置工具
  * 无需额外安装软件
  * 快速易用
  * 快捷键：Windows键 + Shift + R
* **[Mac 截图应用](/docs/recording-work-item-screenshot-app)** - MacOS 内置工具
  * 与 MacOS 深度集成
  * 快捷键：CMD + Shift + 5
  * 无需额外安装软件

### 通用设置流程

1. 在 OneDrive 中创建一个专用文件夹：
   * 对于 Snagit：`OneDrive/Recordings/Snagit/`
   * 对于截图工具：`OneDrive/Recordings/SnippingTool/`
   * 对于 Screenshot App：`OneDrive/Recordings/ScreenshotApp/`
2. 配置你的工具将录制内容保存至相应文件夹
3. 确保麦克风设置正常
4. 测试一次录制以验证设置是否成功
