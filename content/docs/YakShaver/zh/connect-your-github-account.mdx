## 如何将您的 GitHub 账户连接到 YakShaver

1. 如果您使用 Microsoft Teams，进入 Microsoft Teams | YakShaver 机器人 | 在聊天框中输入 'github' 以检查您的 GitHub 状态
   * 如果您尚未连接账户，系统会提示您连接个人 GitHub 账户
   * 如果您已连接账户，您可以看到已连接账户的用户名，并且如果连接出现问题，可以重新连接
2. 在 YakShaver 门户的 [用户设置](https://portal.yakshaver.ai/settings/users) 中，确认您的用户信息中列有您的 GitHub 用户名，否则您可以在那里连接您的 GitHub 项目
3. ✅ 完成！

### 将您的 GitHub 账户连接到 YakShaver 有哪些好处？

1. **问题关联您的 GitHub 用户名** – 问题会显示在您的 GitHub 账户下，所有人无需查看描述即可知道是谁报告的问题。您还将获得此项的 [GitHub 贡献活动](https://docs.github.com/en/account-and-profile/setting-up-and-managing-your-github-profile/managing-contribution-settings-on-your-profile/viewing-contributions-on-your-profile#what-counts-as-a-contribution)！
2. **通知** – 当有人更新或评论您的问题时，您会收到 GitHub 警报。
3. **权限** – 问题遵循您的 GitHub 访问设置。
4. **关注度** – 来自真实用户的问题请求比机器人发起的更受关注。

**注意：** 如果您没有某个待办事项的权限，YakShaver 仍然可以在那里创建工作项，但您将不会获得贡献记录。
