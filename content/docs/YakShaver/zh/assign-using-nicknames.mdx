YakShaver 让您通过为用户使用昵称更自然地分配工作。一旦设置了昵称，它就可以与用户的真实姓名互换——YakShaver 会在提及时识别任意一个。

## 工作原理

1. 设置昵称 - 在门户中，访问 [用户设置](https://portal.yakshaver.ai/settings/users)，选择一个用户，在编辑用户界面添加昵称。每个用户一次只能有一个昵称。
2. 自动检测 - 当录音中提及用户的姓名或昵称时，YakShaver 会检测到该提及并将其链接到正确的用户。
3. 名称互换性 - 一旦设置，昵称在引用时与用户的真实姓名完全相同。您可以使用任意一个分配工作。

![](/zh/YakShaver/Docs/Edit-Nickname.png)**图示：只需添加昵称并在[门户](https://portal.yakshaver.ai)中更新！**

### 示例

* 您将 **Bob Northwind** 的昵称设置为 **Windy**。
* 在一次 shave 中，有人提到：“把这个分配给 **Windy**”
* YakShaver 知道 **Windy** 是 **Bob Northwind**，并相应地分配工作项

### 注意事项

* 昵称不区分大小写
* 每个用户仅限一个昵称
* 对长度或特殊字符无限制
* 您可以随时在同一页面更改或删除昵称
